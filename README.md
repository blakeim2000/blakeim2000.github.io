# Black-Jack
This is a personal project for blakeim2000 to learn the basics of software development principles & tools. 
The aim of this project is to create a single-player Black Jack game with simplified rules.
1. Player starts with 10 chips.
2. Player can bet any number of chips before each round.
3. After getting dealth 2 cards, player can either hit or stay, until they go bust.
4. Dealer will keep picking hit until they reach a number greater than or equal to 16. 
5. If player wins, they double their betted chips, otherwise lose all. 
6. At any given point, the player can "cash out" and submit their name in a global scoreboard based on the number of chips they have. But game will stop after you've "cashed out".  
Base code was taken from https://github.com/ImKennyYip/black-jack.
